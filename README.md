# Amwaj Abayas

## Project Idea
Amwaj Abayas is a web application project that focuses on selling abayas online through a simple and modern website design.

## Problem
Many customers prefer online shopping, but some websites are complicated or not organized clearly when browsing abayas.

## Solution
The proposed solution is to design a web application that allows users to view abaya collections, check product details, and manage their shopping easily.

## Target Users
Women who want to buy abayas online.

## Main Features
- Home page
- View abaya collections
- User profile page
- Shopping cart
- Contact page

## Milestone 1
This repository includes the UI/UX prototype and GitHub setup for Milestone 1.
